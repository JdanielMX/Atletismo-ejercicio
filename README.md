# Bienvenido a mi repositorio


### Ejercicio: Atletismo  

###### Descripción
Pequeño sistema que gestiona atletas (nombre, numero, tiempo)



###### Uso
Debera montar la base de datos para su funcionamiento, en la carpeta util modifique la cadena de conexion de ser necesario.
se incluye la estructura de la base de datos en la raiz del proyecto.

El menu principal cuanta con 3 opciones 

Agregar: permite registrar los datos.
Mostrar: permite ver los registros de los atletas aqui se agregan 3 opciones mas.
- Actualizar: Debe selecionar un atleta de la tabla y dar click en el boton se abre una ventana nueva que permite cambiar los datos.
- Eliminar: Debe selecionar un atleta de la tabla y dar click en el boton.
- Mostrar ganador: muetra los datos del atleta con el menor tiempo(ganador)
